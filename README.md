## Getting Started

### Installation

First, install dependencies:

```bash
npm install
```

### Running the Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Initial Setup - Create Admin User

After starting the application for the first time, you need to set up an admin group and assign it to a user:

```bash
npm run setup:admin
```

This will prompt you to:
1. Enter the email address of the user who should be an admin
2. Enter a name for the admin group (default: "Admins")
3. Enter a description for the group (optional)

**Note**: The user must have logged in to the application at least once before running this command.

For more detailed setup instructions, see [SETUP.md](./SETUP.md).

### Development

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
